Rotates once through a sequence of horizontally-arranged pages, and also allows navigation to an individual page (via SlidingPagesWithDots).

RotatingPagesWithDots seems useful as a home page "hero" element that can present some visually interesting content in hopes the user will find something to engage with. The rotation stops after one cycle to avoid becoming irritating if the user wants to simply read content on the page.

This control was discussed in a user interface blog post.

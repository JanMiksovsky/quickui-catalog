Shows exactly one child at a time. This can be useful, for example, if a given UI element has multiple modes that present substantially different elements.

For a subclass of Modes optimized for controls with two modes for reading and editing, see Editable. The SlidingPanels control is similar to Modes, adding the ability to slide horizontally between pages. The Tabs controls adds a row of tab buttons that govern which child of a Modes control is active.

A panel whose contents will bump up against the top or bottom of a scrolling parent so as to remain always visible.

A PersistentPanel should generally be placed within the highest-level scrolling element on the page; it will not work property when placed in a scrolling element nested within some outer scrolling element.

On mobile touch browsers, the panel may move out of view while the user is dragging the page; the panel will only snap back into view when the user lets go to complete the drag.

This control was discussed in a user interface blog post.
